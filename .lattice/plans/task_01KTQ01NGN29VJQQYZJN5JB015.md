# FRAC-69: Hamburger menu: vertically center with 'Fractal Collective' wordmark + increase size

On mobile the hamburger menu icon is not vertically aligned on the same horizontal axis as the 'Fractal Collective' wordmark in the navbar. It should be — both should sit on the same vertical centerline. Additionally, the hamburger icon feels too small relative to the wordmark; bump up its size for better tap target and visual balance. Scope: (1) locate the navbar component (likely src/components/layout/Navbar.tsx); (2) fix flex alignment so the hamburger and wordmark share a centered horizontal axis (align-items: center on the flex row, ensure no extra padding/margin on one side throwing alignment off); (3) increase the hamburger icon size — pick a size that feels balanced with the wordmark and meets a >= 44px tap target for accessibility. Coordinate with FRAC-60 (hamburger cursor/hover affordance) if implemented in the same pass. Acceptance: at mobile widths, the hamburger icon and 'Fractal Collective' text share the same vertical center; hamburger is visibly larger than current and feels proportionate to the wordmark; tap target is at least 44px.
